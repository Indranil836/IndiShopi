# IndiShopi
---
A E-Commerce site.

# Tech Stack
---
`MERN`
  - nodejs
  - reactjs

We are using docker to containerize it. To replicate this on your local follow the steps:
-
clone this project:
```
git clone <repo_url(HTTP/SSH)>
```
To start this application run:
```
docker compose up
```
To stop this application run:
```
docker compose down
```
